# ActivityGrade-ssm活动评分小程序2.0


## 项目简介

原项目[地址](https://github.com/zan-lab/ActivityGrade-ssm)

本项目为活动评分系统小程序的后端，采用SpringBoot+SpringCloud重构，目前正在进行中。

## 运行条件

* 对应前端支持
* MySQL5.6
* JDK8+
* Tomcat9.0
* 七牛云OSS



## 运行说明

* 小程序部署正常（或者API接口准备好）
* mvn install成功
* jar包运行
* 可选：配置https
